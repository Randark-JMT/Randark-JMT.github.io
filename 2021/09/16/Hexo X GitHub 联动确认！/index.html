Hexo X GitHub 联动确认！

发表于 2021-09-16 更新于 2021-09-18 分类于 其他

1 安装必要的插件

  1. git插件 https://www.git-scm.com/download/
  2. node.js插件 https://nodejs.org/en/
  3. npm插件 包含在node.js插件中
  4. 检测三种插件是否已成功安装：
    1. win+R输入cmd并运行打开命令符窗口
    2. 输入：
    3. git –version
    4. node -v
    5. npm -v
    6. 若均出现版本号则已安装成功

2 搭建GitHub仓库

  1. 注册并登录GitHub后，点击个人头像旁的下三角，下拉菜单中选择Your repositories
  2. 点击绿色的new创建属于自己的仓库
  3. 在创建页面的Repository name栏中输入形同.github.io，其余均非必填项
  4. 点击Create repository创建仓库
  5. 注意事项：repository name务必为.github.io 关系到后期hexo导入

    3 配置ssh密钥

  6. 原因：每次向GitHub提交代码都需要经过你的账号验证，在未配置ssh密钥的情况下，需要每次都输入邮箱和密码，极为繁琐，配置ssh密钥能快速自动通过这一过程，加速代码更新
  7. 查看本机中是否已经存在密钥：
  8. 如果显示No such file则需要重新生成： 邮箱地址应为你的GitHub地址即username.github.io，随后ssh key将保存在C盘用户目录的.ssh文件夹中，路径大致为C:\\Users<你的电脑用户名>.ssh
  9. 右键.ssh文件夹中的id_rsa.pub文件，在打开方式中选择用记事本打开，Ctrl+A全选内容，
    Ctrl+C复制全部内容
  10. 在GitHub中点击右上角的头像，下拉菜单中选择Setting，在左侧菜单中选择SSH and GPG keys
    在顶部右侧选择绿色的New SSH key来新建ssh密钥
  11. 随便起个title，在下方Ctrl+V粘贴你的ssh密钥
  12. 点击Add SSH key完成对ssh密钥的添加
